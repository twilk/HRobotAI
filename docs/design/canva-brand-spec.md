# HRobot — Canva Brand Kit spec (marketing assets)

Drop-in spec for a **Canva Brand Kit** so social posts, decks, and one-pagers match the product.
Canva could not be connected as an MCP in this environment (the connector registry returned no
Canva/design connector), so this is the manual path: paste these values into
**Canva → Brand Hub → Brand Kit**. Pairs with [`DESIGN.md`](DESIGN.md) (the product source of truth).

> Identity to protect: **RODO-native, EU-trust** — precise, secure, EU-grade, with warmth.
> If an asset could have been generated by any AI tool, it is wrong. See the Don't list.

---

## 1. Brand colors → *Brand Kit → Brand colors*

Add these as named swatches (Canva accepts hex):

| Name | Hex | Use in marketing |
|---|---|---|
| Navy (anchor) | `#0B1F3B` | Backgrounds, headlines on light, primary brand block |
| Navy raised | `#15355C` | Gradient pair for the mark/badges |
| Ink | `#101A2B` | Body text on light |
| Canvas (warm) | `#F6F4EE` | Light backgrounds — **not** pure white |
| Signal teal | `#0C8FA3` | One accent: CTAs, underlines, key numbers (use sparingly) |
| Teal on navy | `#3CC3D6` | Accent when the background is Navy |
| Verified green | `#2E9E6B` | "RODO", "szyfrowane", checkmarks, trust ticks |
| Amber | `#B8791F` | Rare attention accent |
| Hairline | `#D9D5C8` | Thin rules, dividers, card borders |

Rule of thumb: **Navy + Canvas do 80%**, teal is a spark (≤10% of any layout), green = trust only.

## 2. Brand fonts → *Brand Kit → Brand fonts*

| Role | Font | In Canva? | How |
|---|---|---|---|
| Display / headline | **Cabinet Grotesk** (700/800) | Upload | Free at fontshare.com/fonts/cabinet-grotesk → upload as Brand Font (Canva Pro) |
| Body / UI | **General Sans** (400–600) | Upload | Free at fontshare.com/fonts/general-sans → upload as Brand Font |
| Mono / technical | **IBM Plex Mono** (400/500) | ✅ Native | Search "IBM Plex Mono" |

**No Canva Pro / can't upload?** Nearest native fallbacks that keep the spirit (still never use
Inter/Montserrat/Poppins): display → **Hanken Grotesk** or **Familjen Grotesk**; body → **DM Sans**;
mono → **IBM Plex Mono**. Document whichever you pick so assets stay consistent.

Type pairing: big Cabinet Grotesk headline (tight tracking, `-2%`) + General Sans body +
short IBM Plex Mono labels (UPPERCASE, wide tracking) for stamps like `RODO · DANE W UE`.

## 3. Logo & mark → *Brand Kit → Logos*

- **Wordmark:** `HRobot` in Cabinet Grotesk 800; `.ai` in **Signal teal** (`#0C8FA3` on light,
  `#3CC3D6` on navy). Tight tracking.
- **Mark:** rounded square (radius ~8–11px) filled Navy→Navy-raised gradient, 1px teal edge,
  containing a smaller rounded-square outline + a centered teal dot (a precise "node" — precision +
  automation, the *robot*). Recreate in Canva with two rounded rectangles + a small circle, or
  export the SVG from `mockups/system.css` (`.mark`).
- **Clear space:** ≥ the height of the mark on all sides. **Min wordmark height:** 20px digital.
- **Don't:** recolor the mark, add glow/shadow, stretch, or place on a busy photo.

## 4. The motif (optional, brand moments)

Faint **engraved diagonal hairlines** (like a passport/certificate guilloché) — a security cue.
In Canva: a thin-line pattern at ~5–6% opacity, 115° angle, over Navy. Use only on hero/cover
panels, never behind body text. (Source: `.motif-navy` in `system.css`.)

## 5. Asset recipes

- **LinkedIn / cover banner:** Navy background + faint motif; Cabinet headline (white) with one
  teal word; a mono stamp `RODO · DANE W UE · SZYFROWANIE AES-256`; mark + wordmark bottom-left.
- **Feature post (square):** Canvas background; Cabinet headline (Navy); one teal underline; a
  small **Ochrona danych**-style list with green ticks (Izolowana baza · Szyfrowanie PESEL ·
  Audyt · Region EU). This is the most on-brand layout — lead with the security receipts.
- **Slide title:** Navy; left-aligned Cabinet title; mono kicker label above it in teal-on-navy.
- **One-pager / PDF:** Canvas; Navy section bands; teal for one number per section; mono for
  specs/IDs; green for compliance ticks. Generous margins, left-aligned, never centered-everything.

## 6. Do / Don't (anti-slop)

**Do:** left-align; let Navy + Canvas carry it; use teal once per layout; show security as content
(ticks, mono specs, EU region); keep generous whitespace; use real Polish copy.

**Don't:** glassmorphism / frosted cards; purple or rainbow gradients; gradient-filled buttons;
neon cyan; Inter / Montserrat / Poppins; 3 icons-in-circles in a row; center everything; drop-shadows
on the logo; stock "diverse team smiling at laptop" hero; "Built for / Designed for" taglines.
